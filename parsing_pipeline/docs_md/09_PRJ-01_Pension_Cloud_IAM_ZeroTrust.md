## PRJ-01: Public Retirement System - Cloud Migration Discovery, IAM &amp; Zero Trust Strategy

Midwestern public retirement system (pension) · 2023 - 2024

| Client         | Midwestern public retirement system (pension)   |
|----------------|-------------------------------------------------|
| Period         | 2023 - 2024                                     |
| Contract value | $1,400,000                                      |
| Meridian role  | Prime contractor                                |

## Summary

Delivered a cloud migration discovery and transformation roadmap together with an Identity &amp; Access Management strategy and a Zero Trust Network Access (ZTNA/SASE) strategy for a public pension system administering multiple defined-benefit and defined-contribution funds.

## Scope Delivered

-  Application assessment and cloud adoption / transformation plan.
-  IAM strategy: identity governance, privileged access, MFA, lifecycle.
-  Zero Trust / SASE reference architecture aligned to NIST 800-207.
-  Compliance certification and mandatory respondent forms completed for the procurement.

## Technologies

AWS, Microsoft Entra ID, Okta, NIST 800-53 / 800-207, SASE.

## Outcomes

-  Board-approved 3-year cloud and Zero Trust roadmap.
-  Reduced standing privileged accounts by 70% in the target design.
